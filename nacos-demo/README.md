# nacos-demo
spring-cloud-nacos

## 一. 项目介绍  
    本项目为nacos动态更新配置以及服务注册发现demo

## 二. 目录结构 
   + com.cn.csx.broker     队列存储器
   + com.cn.csx.consumer   消费者
   + com.cn.csx.producer   生产者 

## 三. 使用说明
   1. 先启动BrokerServer
   2. 启动生产者ProduceClient初始化队列
   3. 启动消费者ConsumerClient消费数据
